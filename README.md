# Intergrity_checker
**A SHA-256 based file integrity monitoring system that detects unauthorized file modifications, additions, and deletions through secure cryptographic hash verification.**
A security-focused file integrity monitoring system that uses SHA-256 cryptographic hashing to detect unauthorized changes within a folder. The application creates a baseline of hash values for all files and continuously verifies their integrity by comparing current hashes against the stored records. It can identify modified, deleted, and newly added files, helping users monitor data integrity and detect potential tampering. This project demonstrates practical applications of cryptography, secure hashing algorithms, and file system monitoring for cybersecurity and digital forensics use cases.

Key Features

* SHA-256 based file hashing
* Folder-wide integrity verification
* Detection of modified, deleted, and added files
* Automated integrity reports
* Secure baseline hash storage
* Simple and efficient monitoring process
